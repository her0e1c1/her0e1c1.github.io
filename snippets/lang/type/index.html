8

type.scm¶

$ ee '(type-of 42)'

integer

$ ee '(type-of 3.14)'

float

$ ee '(type-of "foo")'

string

$ ee '(type-of '\''(1 2))'

cons

$ ee '(type-of '\''(1 2))'

cons

$ ee '(type-of '\''foo)'

symbol

$ ee '(type-of ?a)'

integer

$ ee '(symbolp '\''foo)'

t

$ ee '(symbol-name '\''foo)'

"foo"

$ ee '(symbol-value '\''foo)'
Symbol's value as variable is void: foo

値セルに値があればt、なければnil

$ ee '(boundp '\''foo)'

nil

関数セルに関数があればt、なければnil

$ ee '(fboundp '\''foo)'

nil

$ ee '(symbol-function '\''foo)'

nil
