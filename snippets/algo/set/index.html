complements.scm¶

not (A and B)¶

\[\begin{eqnarray} (A \cup B) - A \cap B \end{eqnarray}\]
def f(A, B):
 return [x for x in A if x in B]
print(f([1,2,3,4,5], [1,3,5,7,9]))

[1, 3, 5]

$ py 'f=lambda a,b: (set(a)|set(b)) -(set(a)&set(b)); P(f([1,2,3,4,5], [1,3,5,7,9]))'
set([9, 2, 4, 7])

union.scm¶

\[\begin{eqnarray} A \cup B = A \cup (B \setminus A) \end{eqnarray}\]

Aの要素は全て使うので、共通部分を除いたB

def f(A, B):
 return A + [b for b in B if b not in A]
print(f([1,2,3,4,5], [1,3,5,7,9]))

[1, 2, 3, 4, 5, 7, 9]

O(N^2)

def f(A, B):
 C = []
 for x in A + B:
  if x not in C:
   C.append(x)
 return C
print(f([1,2,3,4,5], [1,3,5,7,9]))

[1, 2, 3, 4, 5, 7, 9]

$ py 'P(set([1,2,3,4,5]) | set([1,3,5,7,9]))'
set([1, 2, 3, 4, 5, 7, 9])
