index.scm¶

群¶

群とは、以下の3つ満たす演算Xを定めることができる集合G

結合法則

\[\begin{eqnarray} (a X b) X c = a X (b X c) \end{eqnarray}\]

単位元の存在

\[\begin{eqnarray} a X e = e X a = a \end{eqnarray}\]

逆元xの存在

\[\begin{eqnarray} a X x = x X a = e \end{eqnarray}\]

可換群(アーベル群)¶

群の性質に加えて可換も成立

\[\begin{eqnarray} a X b = b X a \end{eqnarray}\]

写像¶

集合AからBへの写像f

\[\begin{eqnarray} f : A \rightarrow B, a \in A, f(a) \in B \end{eqnarray}\]

fが単射

\[\begin{eqnarray} a \ne b \Rightarrow f(a) \ne f(b) \end{eqnarray}\]

fが全射

\[\begin{eqnarray} \forall b \in B, \exists a \in A \quad s.t. \quad f(a) = b \end{eqnarray}\]

全単射は、単射かつ全射 f(x) = ax + bは、全単射 f(x) = e^xは、単射 f(x) = x^2は、f(x)>=0において、全射

sgn¶

+1 or -1

分母は正にできるので、s(i)>s(j)の転倒数が偶数なら1

\[\begin{eqnarray} sgn(\sigma) = \prod_{i,j} \frac{\sigma(i) - \sigma(j)}{i-j} \end{eqnarray}\]
\[\begin{eqnarray} sgn(e) = 1 \end{eqnarray}\]
\[\begin{eqnarray} sgn(互換) = -1 \end{eqnarray}\]

行列式¶

\[\begin{eqnarray} |A| = det A = \sum_{s \in S_n} sgn(s) a_{1s(1)} a_{2s(2)} ... a_{ns(n)} \end{eqnarray}\]
\[\begin{eqnarray} |{}_t A| = |A| \end{eqnarray}\]

多重線形性+交代性(ある列に他の列の定数倍を加えてもよい)

\[\begin{eqnarray} det (a_1, ..., a_i, ..., a_j, ... a_n) = det (a_1, ... a_i + k a_j, ..., a_j, ... a_n) \end{eqnarray}\]
\[\begin{eqnarray} det AB = det A det B \end{eqnarray}\]

vector.scm¶

ベクトル空間¶

+について可換群

\[\begin{eqnarray} \forall x, y \in V \quad s.t. \quad x + y \in V \end{eqnarray}\]

2種分配法則, 結合法則, 1の存在

\[\begin{eqnarray} \forall x \in V, \forall l \in K \quad s.t. \quad l x \in V \end{eqnarray}\]

線形写像¶

V, V’をK上のベクトル空間とし、写像T:V->V’が以下を満たすときTを線形写像と呼ぶ

x, y $in V

\[\begin{eqnarray} T(x+y) = T(x) + T(y) \end{eqnarray}\]

x $in V, l $in K

\[\begin{eqnarray} T(lx) = lT(x) \end{eqnarray}\]

VとV’は同型

\[\begin{eqnarray} Tが全単射 \Leftrightarrow V \cong V' \end{eqnarray}\]

部分空間¶

\[\begin{eqnarray} W \subset V, \forall x,y \in W, \forall m,n \in K \quad s.t. \quad m x + n y \in W \end{eqnarray}\]

Ker, Im¶

線形写像T:V->V’

\[\begin{eqnarray} Ker(T) = \{x \in V \mid T(x) = 0 \} \end{eqnarray}\]
\[\begin{eqnarray} Im(T) = \{T(x) \in V' \mid x \in V \} \end{eqnarray}\]

内積¶

VをR上のベクトル空間とし、Vの任意のベクトルa,bについて、実数(a,b)が定義されるとき、(,)をV上の内積という

対称性

\[\begin{eqnarray} (a,b) = (b,a) \end{eqnarray}\]

線形性(和)

\[\begin{eqnarray} (a_1+a_2,b) = (a_1,b)+(a_2,b) \end{eqnarray}\]

線形性(定数倍)

\[\begin{eqnarray} (la,b) = l(a,b) \end{eqnarray}\]

正値性

\[\begin{eqnarray} (a,a) \ge 0 and (a,a)=0 \Rightarrow a=0 \end{eqnarray}\]
